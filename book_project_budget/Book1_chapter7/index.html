    7.4 Option C: Render¶

    ‚úÖ Pros:

    • Simple, fast deploys

    • Can run background tasks

    üö´ Cons:

    • Cold starts like Railway

    1. Create backend repo ‚Üí push to GitHub

    2. Go to https://render.com

    3. New ‚Üí Web Service ‚Üí Connect your GitHub repo

    5. Set environment variables

    6. Done ‚úÖ
